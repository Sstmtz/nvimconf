# 开关

- 显示行号: <leader> + n
- 显示相对行号: <leader> + rn
- 开启Codeium: <leader> + c

# 光标移动

- 移动到任意位置: s

# 缓冲区

- 开启一个新的缓冲区: <leader> + b
- 关闭当前缓冲区: <leader> + x

# 内置工具

- file tree: <ctrl> + n
- file explorer: <leader> + -
- lazydocker tool: <leader> + ld
- lazygit tool: <leader> + lg
- theme selector: <leader> + th

# 终端

- 打开水平终端: <leader> + h
- 打开垂直终端: <leader> + v

# Git相关

- 查看git状态: <leader> + gt
- lazygit面板: <leader> + lg

# Web集成

- 在浏览器打开选中的url: gs
